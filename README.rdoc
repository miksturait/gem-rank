gem-rank

Aplikacja pozwalająca użytkownikowi:

- logowanie/wylogowywanie
- dodawanie propozycji gemów najczęściej wykorzystywanych/najlepszych zdaniem danego użytkownika wraz z opisem uzasadniającym wybór
- przeglądanie listy gemów
- dodawnie aleternatywnego gemu powiązanego z danym gemem
- ocenianie poszczególnych gemów
- wyszukiwanie gemów
- statystyki popularności gemów na podstawie danych użytkowników github
